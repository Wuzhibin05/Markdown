列表

1，定义格式

  • 有序列表(点后要加空格)
1. 文本1
2. 文本2
3. 文本3
  • 无序列表（-，+，*）

- 文本1
- 文本2
- 文本3

+ test1
+ test2
+ test3


* test1
* test2
* test3
  • 无序列表嵌套
+ 列表一
+ 列表二
    + 列表二-1
    + 列表二-2
    + 列表二-3
+ 列表三
    * 列表一
    * 列表二
    * 列表三
